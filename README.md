# power-query-data-transformation
A collection of Power Query tasks supporting both teaching development and advanced skill building. This repository demonstrates practical ETL workflows (Extract, Transform, Load) for cleaning, shaping, and preparing data in both Excel and Power BI.

Advanced techniques are further developed through structured training with UniTrain Centre, with a strong focus on real-world data scenarios and analytical thinking.

File 1: Fitness Data Cleaning (Fitness_messy)

This file includes a basic hands-on transformation pipeline applied to a messy fitness dataset, demonstrating end-to-end data preparation.

Key Transformations
- Data Type Standardisation (ensure each column responsed to correct data type)
- Understand Null Values (though it is not removed or added in this dataset)
- Duplicate Removal (combine columns to form a "Composite Key" to identify duplicates)
- Replace Values (5 stars => 5, N/A to _...)
- Fix Casing Transformation (via Transform to Capitalise Each Letter)

This repository serves two key purposes:

Teaching Resource
Supports students in understanding structured data transformation workflows and best practices.
Skill Development Portfolio
Demonstrates applied Power Query capabilities in handling messy, real-world datasets.
